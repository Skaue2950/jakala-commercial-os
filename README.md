# GTM OS

This repository contains our go-to-market operating system.

Everything related to sales, positioning, accounts and campaigns lives here.

Folders:

knowledge/
Positioning, market insights, competitive research.

playbooks/
Repeatable GTM strategies.

accounts/
One folder per strategic account.

work/
Active campaigns and projects.

team/
Who owns what internally.

processes/
Operational checklists.

log/
Important decisions and session logs.
